-----How to use-----
1.  Launch and Administrator Rights
    ---------------------------------
    The program will automatically request administrator rights on startup.

2.  Process Selection
    -----------------
    *   Enter the process name or click "Browse".
    *   Critical processes are marked with ⚠ (red), normal ones with ✓.

3.  Value Size Settings
    -----------------------------
    In settings you can select data size: 1, 2, 4 or 8 bytes.

4.  Value Search (First Scan)
    ----------------------------
    *   Enter a number and click "First Scan".
    *   You can enable the digit count filter.

5.  Search Refinement (Next Scan)
    ------------------------------
    *   Change the value in the game, enter the new value or select comparison type.
    *   Click "Next Scan". The list will update.
    *   Enable "Fast mode" for faster Next Scan.

6.  Working with Addresses
    -------------------
    *   Select an address in the list, click "Read" to check.
    *   Enter a new value and click "Write".

7.  Error Handling
    -------------------
    *   On crash, the program will create a crash_log.txt file.
    *   There is an "Open Logs" button in settings.
